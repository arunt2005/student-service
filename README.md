# Student Service

A Spring Boot application to manage student records with CRUD operations, built using Java 21.

## 🔧 Tech Stack
- Java 21
- Spring Boot
- Spring Data JPA
- H2 Database
- MapStruct (for DTO mapping)
- Gradle
- IntelliJ IDEA

## 🚀 Getting Started

### Prerequisites
- Java 21+
- Gradle
- IntelliJ IDEA (or any IDE)

### How to Run
```bash
./gradlew bootRun
